# Bitburner v2

A from-scratch rewrite of my Bitburner automation scripts, built on a new save game.

## Goals

- Clean, well-organized scripts written incrementally, without copying code from the previous attempt
- Use the in-game docs (`docs/NetscriptDefinitions.d.ts`, game guides) as the only reference
- Document the reasoning behind each script as it's built — see `docs/devlog/`

## Structure

- `src/` — Netscript scripts (the actual code that runs in-game)
- `docs/` — design notes, devlog entries, and reference material
