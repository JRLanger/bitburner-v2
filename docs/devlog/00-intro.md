# Devlog 00 — Starting Over

**Date:** 2026-06-15

## Why

Previous Bitburner scripts grew organically and got hard to follow. Starting a new save game and rewriting everything from scratch, using only documentation (not old code) as reference, with the goal of being more organized from day one.

## Plan

- Set up git/GitHub for version control
- Build scripts incrementally, smallest useful piece first
- Write a devlog entry for each meaningful script/decision: what it does, why it's built that way, and what alternatives were considered
